Welcome to AWS Logging Handlers’s documentation!¶

aws_logging_handlers is an open source package that was written to provide a seamless and abstract way to emit log records to the cloud.

It currently supports AWS S3 and Kinesis Streams services, additional services/vendors will be evaluated upon community demand.
